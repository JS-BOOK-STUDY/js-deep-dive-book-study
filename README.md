## 1. git clone, commit, push
```sh
# 리포지토리 클론하기
git clone https://github.com/JS-BOOK-STUDY/js-deep-dive-book-study.git

# main 브랜치에서 자신의 닉네임으로 브랜치 생성 및 체크아웃
git checkout -b {닉네임}

# 커밋 메세지 작성
git commit -m '[{닉네임}] week1 정리' # 커밋 메세지 예시 : git commit -m '[sian] week1 정리'

# 자신의 닉네임 브랜치로 푸쉬
git push origin {닉네임} 
```
